# tgbox-cli-build

This repository was designed for making [**tgbox-cli**](https://github.com/NotStatilko/tgbox-cli) builds for **Linux**, **Windows** and\
**MacOS**, with fast `cryptography` and [**FFmpeg**](https://github.com/NotStatilko/ffmpeg-preview) included.

We create executables with [**Github Actions**](https://docs.github.com/en/actions). Any change in the `config.env`\
will **trigger build process** for all machines. You can get executables in [**Releases**](https://github.com/NotStatilko/tgbox-cli-build/releases).
